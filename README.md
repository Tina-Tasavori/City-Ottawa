# City-Ottawa
Infrastructure visualization dashboard for City of Ottawa — ward-level maps and asset lifecycle insights
# Infrastructure Visualization for Ottawa City Councilors

**Client:** City of Ottawa  
**Institution:** Carleton University  
**Tools:** Python · ArcGIS Pro · Power BI

## Overview
A dashboard and ward-level map system that consolidates 
municipal infrastructure data to support evidence-based 
decision-making by city councilors.

## Features
- Interactive ward-level condition maps
- Asset lifecycle dashboards (roads, bridges)
- Renewal priority scoring
